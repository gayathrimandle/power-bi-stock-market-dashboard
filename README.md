# Stock Market Performance Dashboard (Power BI)

## Project Overview

This project presents a **Stock Market Performance Dashboard** built using **Power BI** to analyze stock price trends, trading volume, and monthly performance patterns.
The dashboard provides interactive insights into stock behavior over time using visual analytics.

---

## Dashboard Features

* KPI cards displaying:

  * Average Closing Price
  * Highest Stock Price
  * Lowest Stock Price
* Stock Price Trend Analysis
* Trading Volume Visualization
* Monthly Average Closing Price Analysis
* Interactive Year Slicer for filtering the data
* Multi-page dashboard for structured analysis

---

## Dashboard Pages

### 1. Overview

Provides a high-level summary of stock performance including key metrics and overall trends.

### 2. Monthly Analysis

Shows the average closing price by month to identify seasonal patterns in stock performance.

### 3. Insights

Summarizes key findings derived from the dashboard analysis.

---

## Key Insights

* The stock shows an overall upward trend across the analyzed period.
* Significant trading volume spikes occur during major price movements.
* Certain months demonstrate higher average closing prices, indicating possible seasonal trends.
* Interactive filters allow users to explore yearly stock performance.
